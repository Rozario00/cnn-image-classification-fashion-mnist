# Project Overview

This project builds a CNN-based image classification model capable of identifying clothing items from grayscale images.

The model learns visual features using convolutional layers and predicts one of **10 clothing categories**.

---

# Dataset

Fashion-MNIST dataset

- Training images: **60,000**
- Test images: **10,000**
- Image size: **28 × 28 grayscale**
- Classes: **10 clothing categories**
